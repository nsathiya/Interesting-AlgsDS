# Interesting-AlgsDS

```
1. cd Problems
2. javac <PROBLEM_NAME>
3. java Solution
```

1. De/Serialize N-ary Tree (BFS approach)
2. Longest Substring Without Repeating Characters (DP approach)
3. Find Longest Path Visited In DAG (Topological sort- no weighted edges)
